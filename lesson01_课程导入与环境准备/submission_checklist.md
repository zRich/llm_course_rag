## 🚀 功能测试步骤

### 1. 应用启动测试
```bash

# INFO:     Application startup complete.
```

### 2. 接口功能测试
```bash

curl http://localhost:8000/

# 预期响应:
# {"message":"欢迎使用RAG课程API"}

# 测试健康检查
curl http://localhost:8000/api/v1/health

# 预期响应包含:
# {"status":"healthy","message":"RAG系统运行正常","timestamp":"..."}
```

- 访问 http://localhost:8000/redoc (ReDoc)
- 确认文档正常显示且可以测试接口

uv run pytest

### 5. 代码质量检查
